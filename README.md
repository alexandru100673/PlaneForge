# PlaneForge

PlaneForge is an open-source application that transforms scanned RC aircraft plans and PDF drawings into CNC-ready kit files.

## Vision

The goal of PlaneForge is to become the most advanced open-source toolkit for converting traditional RC model plans into manufacturable laser-cut kits.

## Main Features

- Import PDF drawings
- Import scanned plans
- Automatic page alignment
- Automatic contour detection
- Rib detection
- Spar detection
- Former detection
- Automatic vectorization
- Finger joint generation
- Dovetail joint generation
- SVG export
- DXF export
- LightBurn export
- G-code preparation

## Design Philosophy

- Documentation first
- Small independent modules
- Every Python file under 500 lines
- Test before merge
- Clean architecture
- Open source
